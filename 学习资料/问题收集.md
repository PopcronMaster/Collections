#### 清屏快捷键ctrl+k失效

```
ctrl+L` 清屏只是调至当前行
 `ctrl+k` 真正清屏，失效时作用同`ctrl+L
```

按以下步骤修正：【其他快捷键方法雷同】

> **step1:**
>  依次打开  `文件 > 首选项 > 键盘快捷方式 > keybindings.json`

> **step2:**
>  右侧编辑json区域加入以下代码

```
{
    "key": "ctrl+k",
    "command": "workbench.action.terminal.clear",
    "when": "terminalFocus"
  }

```
